# **App Name**: TelMarq

## Core Features:

- Secure Google OAuth Login: Securely authenticate users with Google OAuth before accessing content using GOOGLE_CLIENT_ID from .env.
- Browse Phone Listings: Display available mobile phones for sale in a grid or list format.
- Detailed Product Pages: Show detailed information about each phone, including specifications, condition, and seller details.
- Buy Now Functionality: Enable users to purchase a phone via a 'Buy Now' button, initiating a purchase process.

## Style Guidelines:

- Primary color: Muted teal (#74C69D) for a calm and reliable feel.
- Background color: Soft white (#F0F7F4) for a minimalist, clean look.
- Accent color: Light sea green (#A8D5BA) to complement the primary color with slight contrast.
- Body and headline font: 'Inter', a sans-serif font providing a modern and clean aesthetic.
- Simple, outline-style icons to represent phone features and categories, in muted teal.
- Clean and minimalist layout with plenty of whitespace for a focused user experience.
- Subtle transitions and animations to enhance user interaction and provide visual feedback.
